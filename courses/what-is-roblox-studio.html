1. Introduction to Roblox Development

What is Roblox Studio?

Roblox Studio is the game engine used by Roblox game developers to create and publish games, formerly known as experiences, on the Roblox Platform. (You cannot use Roblox Studio to publish a game outside of the Roblox platform)

Unlike most of the other popular game engines out there, Roblox provides servers for your games without any fees, which means you can create a multiplayer game without needing to pay to run it.
